# Monkey Electron App Icon

Place `icon.ico` (256x256, Windows) and `icon.icns` (macOS) here.

## Quick method

1. Create a 256x256 PNG icon
2. Install png2icons: `npm i -g png2icons`
3. Run: `png2icons icon.png . -ico`
4. Or use https://icoconvert.com to convert PNG → ICO

## Design

Monkey emoji 🐵 on orange gradient background, matching the macOS app icon.
